# Iris Flower Classification

## 📌 Project Description
The Iris Flower Classification project is a Machine Learning application that classifies iris flowers into one of three species using their flower measurements. The model is trained on the Iris dataset and predicts the flower species with high accuracy.

## 🎯 Objective
To develop a machine learning model that accurately classifies Iris flowers based on sepal length, sepal width, petal length, and petal width.

## 🌸 Iris Flower Species
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## 🛠 Technologies Used
- Python
- Scikit-learn
- NumPy
- Matplotlib

## 📂 Dataset
The project uses the built-in Iris dataset provided by the Scikit-learn library.

## 🚀 Features
- Loads the Iris dataset.
- Splits the dataset into training and testing sets.
- Trains a Decision Tree Classifier.
- Predicts the species of Iris flowers.
- Calculates and displays model accuracy.
- Visualizes the accuracy using a bar chart.

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the Python file:

```bash
python iris_flower_classification.py
```

## 📊 Sample Output
- Displays the prediction results.
- Shows the model accuracy.
- Generates an accuracy bar chart.
